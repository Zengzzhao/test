# Weighted Linde-Buzo-Gray Stippling

加权LBG点刻

介绍了一个新的点分布算法，用于生成高质量的点刻图、采样点集、网格重建

## 历程

LBG三人提出的向量量化LBG算法（通过动态分裂Voronoi单元，直到达到所需的代表性向量数量）

基于Voronoi单元的大小、灰度值、输入图像的方差重新定义该算法的分裂与合并过程。该方法自动适应各种约束条件。与先前不同：不需要好的初始点分布/对最终点数的先验知识。

与加权Vorinoi点刻相比，加权LBG点刻收敛速度更快，光谱和空间特效更好。

点的创建基于局部操作，所有可以生成一致连续的点刻动画

## 研究背景

在计算机图形学中，很多应用（如非真实感绘制、采样、网格重建）都需要“分布均匀但密度可控”的点集。传统的 **Lloyd算法**（基于Voronoi图的迭代优化）常用于此目的，但存在以下问题：

- 收敛慢，尤其当初始点分布不佳；
- 需要提前知道点的数量；
- 最终可能形成不自然的六边形结构；
- 不易动态调整或动画化。

> Voronoi图（沃罗诺伊）：一种把平面划分为若干区域的几何结构，每个区域包含距离某个点（生成点/种子点）最近的所有位置
>
> Lloyd's算法（劳埃德）：一种基于Voronoi图的点分布优化算法，用来让点集分布得更加均匀符合目标密度。主要思想是给定一组点，计算它们的Voronoi图，把每个点移动到自己的Voronoi区域的质心，重复迭代多次，直到点位置逐渐趋于平衡
>
> LBG算法：由Linde、Buzo、Gray三人提出，主要思想是随机初始化若干中心点，把数据分配到离自己最近的中心，更新每个中心为其分配样本的均值，判断误差是否收敛，否则重复，若误差太大则分裂部分中心，使得点数量变多，继续优化。类似Lloyd算法，不过Lloyod算法点数固定，而LBG可以通过合并分裂动态增加减少点
>
> Weighted LBG算法：LBG的加权版本，用于在非均匀密度下生成点分布。主要思想是使用Voronoi图划分空间，对每个Voronoi区计算目标密度，若区域密度太高则分裂、若密度太低则合并，迭代更新得到与目标密度匹配的点分布，同时可以调整每个点的大小

## 论文创新点

提出了weighted LBG算法的改进版-->weighted LBG stippling。

**动态分裂与合并 Voronoi 单元**

- 根据输入图像的灰度值（或其他密度函数），算法自动决定是否在某一区域“增加点”或“减少点”；
- 实现从一个随机初始点出发，逐步生成目标点集，无需预设数量。

**自适应密度控制**

- 点的分布根据图像灰度或几何特征自动变化；
- 可以生成不同点大小或密度变化的艺术化点描。

**高收敛速度**

- 相比传统的加权Lloyd算法，LBG-Stippling 收敛更快；
- 只需较少迭代即可得到高质量分布。

**支持动画与三维应用**

- 因为操作是局部的（split/merge），能保持帧间一致性，生成连贯的点动画；
- 也可用于3D网格重采样（remeshing），按曲率自适应调整顶点密度。

## 算法细节

1. **输入**：灰度图或几何模型；
2. **计算Voronoi图**；
3. **根据每个单元的灰度总量：**
   - 小于下阈值 → 删除点；
   - 大于上阈值 → 分裂单元；
   - 介于两者 → 保留并移动到质心；
4. **迭代执行，直到稳定**。



# Stippling of 2D Scalar Fields

二维标量场的点刻可视化

利用点刻可视化二维连续/离散数据

## 研究背景

传统的二维标量场（如地形高度图、人口密度、温度分布等）常用颜色映射/等高线来表现，缺点是

- 颜色感知容易受光照、设备等影响；
- 等高线虽可表达结构，但细节较少；
- 图形往往难以兼顾“整体结构”与“局部细节”。

点刻是一种以密度和点大小传递信息的可视化抽象

- 无需颜色即可表达连续变化；
- 兼具整体感和细节层次；
- 艺术化与数据表达兼容。

作者提出将点刻从艺术渲染扩展到科学与信息可视化领域。

## 贡献

基于Weighted LBG Stippling算法（原本用于图像抽象化和网格重采样），提出针对二维标量场的改进算法，主要贡献包括：

1. 将 LBG 算法推广到连续与离散标量场；
2. 可控的点大小与密度调节机制，可在视觉上更灵活地表达数据；
3. 引入两种基于点分布的“轮廓模拟”方法：
   - Restricted Stippling（约束点刻）：通过限制点在等值区间内移动来形成“轮廓线”；
   - Mach-Banding（马赫带点刻）：借助视觉对比增强效应强化边界；

## 方法原理

**基础算法：LBG**

- 基于 Voronoi 图，将平面划分为若干单元；
- 每个点迭代移动至自身 Voronoi 区的加权质心；
- 根据目标密度函数决定是否分裂/删除点；
- 迭代至稳定，形成均匀且符合数据分布的点集。

**针对标量场的改进**

- 输入数据为连续或离散的标量函数 φ(x,y)；
- 通过映射函数 ρ(φ) 将其归一化为密度 Φ(x,y)；
- 每个点的目标“墨量”通过积分得到；
- 点的大小与密度可分别由 Φ 控制，从而在**点大小**与**点密度**两个视觉通道中独立编码信息。

**轮廓模拟方法**

- Restricted Stippling约束点刻：将点限制在特定等值区间内，边界处留白 → 自然形成“负空间”轮廓；
- Mach-Banding马赫带点刻：利用高斯模糊与差分增强边界区域的对比度，从而产生视觉上的轮廓强化。

## 实验与案例

作者将该方法应用到多种类型的数据上，其中一个实验案例就是散点图数据，用点刻替代传统散点图，有效缓解过度绘制问题

## 流程

密度标量场构建：使用核密度估计方法将离散的二维数据点转变为连续的密度标量场，作为下一步的输入。

自适应蓝噪声采样：以密度标量场作为输入，以可视表征标量场为目的，在可视空间执行采样，构造一组空间频率不同的采样点（刻点），从而实现点刻。从个体角度看，数据点分布随机且均匀，从整体角度看，数据点分布精准重建了密度场的密度分布。至此，实现了对散点图的可视抽象。

采样点染色：对每个采样点，根据其所在位置附近的真实数据点的类别来染色，从而保证刻点体现的类别分布与原始数据点的类别分布一致。这就实现了对多类散点图的可视抽象。

交互式可视增强：提供一个可视增强工具，用户在该工具中可根据具体的分析需求和喜好，调整灰度-密度曲线，实现对密度感知的任意调整。比如，通过增强全局的密度对比度凸显团簇，通过增大低密度区域刻点的大小凸显离群点。
